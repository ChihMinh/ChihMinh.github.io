将项目chatroom push到GitHub仓库中去。Git文件有三种状态，已修改、已暂存、已提交，Git工作流程：
a. 在工作目录中修改某些文件。
b. 对修改后的文件进行快照，然后保存到暂存区域。
c. 提交更新，将保存在暂存区域的文件快照永久转储到 Git 目录中。
